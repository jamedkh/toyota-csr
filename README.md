# TW Associates Website (TW1.com)

> Pakistan's Premier Tier 1 Private Sector Submarine Fiber Optic Cable Network Provider

![Homepage Screenshot](tw1_screenShot.png)

## 🌐 Overview

This is the official website for TW Associates (Transworld), Pakistan's leading telecommunications infrastructure provider. The website showcases comprehensive internet connectivity solutions, data center services, and enterprise networking solutions.

## 🚀 Features

- **Responsive Design** - Optimized for all devices and screen sizes
- **Modern UI/UX** - Clean, professional interface with intuitive navigation
- **Service Portfolio** - Comprehensive display of telecommunications services
- **Performance Optimized** - Fast loading times and smooth animations
- **SEO Friendly** - Structured markup and optimized content
- **Cross-browser Compatible** - Works seamlessly across all modern browsers

## 🛠️ Technology Stack

- **HTML5** - Semantic markup and modern web standards
- **SCSS** - Advanced CSS preprocessing with variables and mixins
- **Bootstrap** - Responsive grid system and UI components
- **Gulp.js** - Build automation and asset optimization
- **JavaScript** - Interactive functionality and animations

## Quickstart guide

- Clone or download this Git repo onto your computer.
- Install [Node.js](https://nodejs.org/en/) if you don't have it yet.
- Run `npm install`
- Run `gulp bs` to run the default Gulp task

In this proejct, Gulp is configured to run the following functions:

- Compile the SCSS files to CSS
- Autoprefix and minify the CSS file
- Purging css file (seperate folder)
- Concatenate the JS files (in order we have give for compatibility)
- Uglify the JS files
- Moving assets to `/dist` folder
- Minify output html files in `/dist` folder
- Move final CSS and JS files to the `/dist` folder
